# BLAZE

## Installation and Deployment

1. **Clone repository:**

   ```bash
   git clone https://github.com/181charan/BLAZE-SC-ad
   ```

2. **Build BLAZE executable:**

   ```bash
   ./build.sh
   ```

3. **Run performance tests:**

   ```bash
   ./perf_test.sh
   ```

4. **Generate result graphs:**

   ```bash
   ./generate_graphs.sh
   ```
